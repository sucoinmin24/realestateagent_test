# 공인중개사 시험 문제집 PWA

## 📱 아이패드에 설치하는 방법

### 1단계: 파일 호스팅
이 파일들을 웹 서버에 올립니다 (무료 옵션):

**GitHub Pages (추천)**
1. GitHub에 새 repository 생성
2. 모든 파일 업로드
3. Settings → Pages → Branch를 'main'으로 설정
4. 제공된 URL 복사 (예: https://username.github.io/realtor-exam/)

**다른 옵션**
- Netlify (netlify.com)
- Vercel (vercel.com)
- Firebase Hosting

### 2단계: 아이패드에서 설치

1. **Safari 브라우저**로 호스팅된 URL 접속
2. 화면 하단의 **공유 버튼** (⬆️) 탭
3. **"홈 화면에 추가"** 선택
4. 앱 이름 확인 후 **"추가"** 탭
5. 홈 화면에 앱 아이콘 생성됨!

### 3단계: 앱처럼 사용
- 홈 화면의 아이콘을 탭하면 전체 화면으로 실행
- 오프라인에서도 작동
- 일반 앱과 동일한 사용 경험

## 🔄 자동 업데이트 방법

### 파일 업데이트 시:
1. GitHub (또는 호스팅 서비스)에서 파일 수정
2. 변경사항 저장/푸시
3. **사용자가 앱을 다시 열면 자동으로 최신 버전 다운로드**
4. 추가 작업 필요 없음!

### Service Worker가 자동으로:
- 새 버전 확인
- 백그라운드에서 다운로드
- 다음 실행 시 자동 적용

## 🔧 문제 추가 방법

`realtor_pwa.html` 파일의 `questionDatabase` 배열에 문제 추가:

```javascript
{
    year: 2025,
    subject: 'intro',  // intro/civil/broker/public/disclosure
    question: "문제 내용",
    options: [
        "선택지 1",
        "선택지 2",
        "선택지 3",
        "선택지 4",
        "선택지 5"
    ],
    correct: 0,  // 정답 인덱스 (0-4)
    explanation: "해설 내용"
}
```

## 💡 팁

### 오프라인 사용
- 첫 실행 후 자동으로 캐시
- 인터넷 없이도 작동
- 비행기 모드에서도 사용 가능

### 업데이트 확인
- 앱을 닫았다가 다시 열면 자동 체크
- 백그라운드에서 자동 다운로드
- 사용자 개입 불필요

### 데이터 관리
- 문제 데이터는 앱 내부에 저장
- 진행 상황은 세션별로 초기화
- 개인정보 수집 없음

## 🆚 APK vs PWA

| 항목 | APK (안드로이드) | IPA (iOS) | PWA |
|------|------------------|-----------|-----|
| 개발 비용 | 높음 | 매우 높음 | 무료 |
| 배포 방법 | Play Store | App Store | URL 공유 |
| 승인 필요 | 필요 | 필요 | 불필요 |
| 업데이트 | 수동 다운로드 | 수동 다운로드 | 자동 |
| 크로스 플랫폼 | ❌ | ❌ | ✅ |
| 개발자 계정 | 필요 ($25) | 필요 ($99/년) | 불필요 |
| 설치 | 복잡 | 매우 복잡 | 간단 |

## 📞 문제 해결

### 홈 화면에 추가가 안 보여요
- Safari 브라우저를 사용하세요 (Chrome ❌)
- iOS 14.3 이상 필요

### 업데이트가 안 돼요
1. 앱 완전히 종료
2. Safari에서 원본 URL 접속
3. 새로고침 (⌘+R)
4. 다시 앱 실행

### 오프라인이 안 돼요
- 앱을 한 번 이상 실행해야 함
- 네트워크 연결 상태에서 첫 실행 필요
